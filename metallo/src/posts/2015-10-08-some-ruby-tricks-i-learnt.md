## How to split an array into slices

```
slices = array.each_slice(n)
```

## Get a random number of element out of an array

```ruby
array.sample(n)
```

## Filter an array

```ruby
array.select { |element| element > 10 }
```

## Get last pash component

```ruby
uri.split('/').last
```
